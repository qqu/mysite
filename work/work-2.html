Ag Data Commons (ADC) is a USDA project providing access to a wide variety of open data relevant to agricultural research.

It's a Drupal site based on dkan service. User interface is built with Drupal panel and bootstrap. We conducted multiple usability tests to collect data and feedback. The design is driven by analysis of the test results.
